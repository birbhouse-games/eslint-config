# @trezy-studios/eslint-config

## Install

### npm
```sh
# Install ESLint and this configuration
npm install -D eslint @trezy-studios/eslint-config

# Install peer dependencies for this configuration
npm install -D @typescript-eslint/parser @typescript-eslint/eslint-plugin eslint eslint-plugin-editorconfig eslint-plugin-import eslint-plugin-jsdoc eslint-plugin-optimize-regex eslint-plugin-promise eslint-plugin-security eslint-plugin-sort-class-members eslint-plugin-unused-imports
```

### Yarn
```sh
# Install ESLint and this configuration
yarn add -D eslint @trezy-studios/eslint-config

# Install peer dependencies for this configuration
yarn add -D @typescript-eslint/parser @typescript-eslint/eslint-plugin eslint eslint-plugin-editorconfig eslint-plugin-import eslint-plugin-jsdoc eslint-plugin-optimize-regex eslint-plugin-promise eslint-plugin-security eslint-plugin-sort-class-members eslint-plugin-unused-imports
```
